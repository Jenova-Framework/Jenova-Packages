# Projekt J.E.N.O.V.A : Packages Distribution


This repository includes official packages prepared for Jenova Framework.

# Package Categories
- **Compilers:** Packages containing prepared minimalistic compilers for the Jenova build system.
- **Godot Kits:** Precompiled GodotSDK packages prepared for integration and communication with the engine.
- **Libraries:** Precompiled static libraries for the Jenova ecosystem. Libraries are provided as .lib, .a and .hpp
- **Sample Projects:** Premade Ready-to-Use projects utilizing Jenova framework and its features, prepared for learning and benchmarks.
- **Code Templates:** Packages containing code templates and smart templates for the framework.
- **Addons:** Precompiled plugins to integrate directly into the framework. Addons come as dynamic libraries.
- **Tools:** Precompiled plugins for the editor, mixing the Extension API and Jenova Framework seamlessly.
